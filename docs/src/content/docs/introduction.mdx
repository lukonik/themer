# Introduction

Themer is a theme management library for React apps built around TanStack Router and TanStack Start.

It gives you a `ThemeProvider` and `useTheme` hook for light, dark, and system themes, with SSR-safe behavior and built-in flash prevention.

## What It Solves

- Applies the correct theme before React hydrates
- Persists theme state with `localStorage`, `sessionStorage`, or cookies
- Resolves `system` to the active OS preference
- Syncs theme changes across browser tabs
- Supports `class`, `data-*`, or multiple attributes at once
- Lets you force a theme for a route or page

## How It Works

`ThemeProvider` manages the theme state and writes the active theme to `document.documentElement`.

For built-in storages, it also renders an inline script before hydration so the `<html>` element already has the correct class or data attribute when the page paints.

That combination is what prevents FOUC and keeps SSR output aligned with the client.

## Core Pieces

- `ThemeProvider`: owns the theme state and applies it to the DOM
- `useTheme`: reads and updates theme state from React components
- `ThemeScript`: injected internally by `ThemeProvider` for built-in storages
- Storage adapters: `localStorage`, `sessionStorage`, `cookie`, or a custom `ThemeStorage`
